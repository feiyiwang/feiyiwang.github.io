My passion for data science stems from my study in Data Science at Worcester Polytechnic Institute (WPI). There I completed required courses such as Statistical Methods for Data Science, Database Applications Development, Big Data Analytics, as well as several advanced practical projects, in which I applied the machine learning methods I learned from class and data visualization tools I learned myself to the real world data. In one of the projects, the organization I worked for wanted to have a dashboard to understand the insights of their conference data. I used Shiny, an R package, and D3.js to build an interactive dashboard for them to view descriptive statistics as well as two predictive models to forecast future conference attendees, a topic model to extract abstract keywords from attendee comments and a clustering model to calculate correlations between each paired keywords.
